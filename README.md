# Bài Tập Kiểm Thử Phần Mềm - Tuần 1

## Mô tả

Chương trình giải phương trình trùng phương: ax⁴ + bx² + c = 0

## Files trong project:

- `phuongtrinhtrungphuong.cpp`: Code chính giải phương trình
- `test.ipynb`: File test cases kiểm tra code
- `README.md`: Tài liệu hướng dẫn

## Cách chạy:

1. Compile: `g++ -o phuongtrinhtrungphuong.exe phuongtrinhtrungphuong.cpp`
2. Chạy: `./phuongtrinhtrungphuong.exe`
3. Nhập 3 số a, b, c
